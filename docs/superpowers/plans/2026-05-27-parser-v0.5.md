> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 LociFind intent-parser 在 v0.5 evals (500 case) 上的 variant confusion 从 51 fail 降到 ≤ 10，纯 parser 规则改进，零模型介入。

**Tech Stack:** Rust 1.95 stable + workspace + `bash scripts/ci.sh`（fmt + clippy `-D warnings` + build + test）+ `packages/evals` v0.5 fixture (500 case)。

/// 视频/图片 类型词的路由判定（v0.5 反转 v0.4 决策）：
///
/// - **具体 size 阈值（"100MB" / "1GB"）→ media_search**（v0.5 fixture 契约）
/// - **抽象 sort 词（最大/largest/最重）→ media_search**
/// - **仅时间词（修改/上周/modified/last week）→ file_search**（v0.5 fixture 契约）
///
/// v0.4 当时选了"具体 size → file_search"，v0.5 全集 fixture 数据（500 case）显示
/// 契约相反；同时拆 ABSTRACT_MODIFIERS 为 SIZE_SORT_WORDS 与 TIME_MODIFIERS，让
/// time-only query 不再误触发 media_search。
fn has_visual_media_with_abstract_modifier(lower: &str) -> bool {
    const VISUAL_MEDIA: &[&str] = &[
        "视频", "video", "videos", "影片", "movies", "图片", "images", "pictures",
    ];
    if !VISUAL_MEDIA.iter().any(|w| lower.contains(w)) {
        return false;
    }
    // v0.5 反转：具体 size 阈值 → media_search
    if has_explicit_size_threshold(lower) {
        return true;
    }
    // v0.5 抽象 size/sort 词 → media_search
    const SIZE_SORT_WORDS: &[&str] = &[
        "最大",
        "最小",
        "最重",
        "最沉",
        "biggest",
        "largest",
        "smallest",
        "体积最大",
        "大文件",
    ];
    if SIZE_SORT_WORDS.iter().any(|w| lower.contains(w)) {
        return true;
    }
    // v0.5 仅时间词 / "最新/最旧" → 返 false，让 file_search 接手
    // (TIME_MODIFIERS 不再触发 media_search)
    false
}
```

pub(crate) fn has_keyword_like_signal(lower: &str) -> bool {
    // v0.5 扩 stop：含 "recent" / 中文虚词，让 "find recent" / "找 recent 的"
    // 走 is_recent_only_query → Clarify
    let stop: &[&str] = &[
        "找", "查找", "最近的", "最近", "the", "show", "me", "find", "recent", "的", "里",
    ];
    let cleaned: String = stop
        .iter()
        .fold(lower.to_owned(), |acc, s| acc.replace(s, " "));
    cleaned.split_whitespace().any(|t| t.len() >= 2)
}
```
