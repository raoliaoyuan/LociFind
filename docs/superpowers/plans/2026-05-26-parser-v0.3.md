## Design Decisions (decided before writing tasks)

1. **Location hint 按输入语言保留**：`parse_location` 当前强制归一到中文（`"下载"`）。fixture 显示 expected 在英文 query 上要求英文 hint（`"downloads"`），中文 query 上要求中文 hint（`"下载"`）。因此 `parse_location` 改为接收 `language: Language` 参数，对 zh/mixed/unknown 输出中文 canonical，对 en 输出英文 canonical。其他可选项（统一中文 / 加 canonical 枚举）会拉大 fixture 改动面，不采用。
2. **`extract_target_ref` 用 regex 通用化**：当前硬编码 1-5 + 仅认 "第 N 个" 含空格 + 仅认 ordinal word "first"。改为正则 `第\s*(\d+)\s*个?` 与 `the\s+(\d+)\s+result`，配合保留原有"第一个/the first" ordinal-word 路径作为 fallback。这是行为扩展，不破坏现有测试。
3. **Refine `try_parse_refine` 加 time / english-only / clear / limit-to 处理**：现状仅识别 `只看/show only` + extension + location + sort + `不限制`。fixture 显示要支持 `只看上周修改的`（time）、`only in X`（英文 only）、`clear the X limit`（英文 clear）、`limit to last week`（英文 time）、`exclude videos`（英文 exclude 已识别但匹配失败 — 因为 lexicon 的 "videos" 在 EXTENSION_ALIASES 列表里没有，需补）。这些都直接落在现有函数里加分支。
4. **Clarify 文案策略**：fixture 把 `question` 和 `options` 字段当字符串数组精确匹配。修改有两条路：
   - (a) parser 文案对齐 fixture（损失中文用户友好度，且 fixture 内自身已有中英混杂的"标准文案"两种风格 — 例如 v05-clarify-template-498 期望中文 options `["全盘搜索","下载","文稿","桌面"]`，v05-clarify-template-491 期望英文 options `["all files","downloads","documents","desktop"]`）。
   - (b) evals 比较器对 Clarify 变体改为 `reason` 精确 + `question`/`options` 弱匹配（语义级，例如 `options.len()` 一致 + 内容做 normalize 比较）。
   策略：**采用 (b) 的轻量版**。`packages/evals/src/lib.rs` 的 `compare_json` 加一个 Clarify-aware 分支，对 clarify case 只比较 `reason` 与 `language`/`schema_version`，并在 `options` 字段上做长度一致检查（不比内容文本）。**问 user 确认再做** — 否则本 plan Task 7 跳过 clarify 修复，只在最终报告里记录"clarify partial 由 evals 比较器层面解决，留 v0.4 或单独 PR"。
5. **不动 MVP-17 fallback 路径**：Class D 36.6% 留给 MVP-17 端到端 evals（STATUS 下一步候选 #2）。本 plan 只动规则路径。
6. **不重构 lib.rs 拆分**：1227 行的 lib.rs 已经接近"该拆"的边界，但本 plan 是 evals 驱动的精确修复，不做架构重构。如果 task 6 后 lib.rs 还在涨，留一个 follow-up "parser v0.4 拆分"task 给后续会话。

```rust
/// 按字符统计判断语言。zh+常见 ASCII 技术名词（文件扩展名 / 大小单位 / 已知短词）
/// 不再触发 mixed —— 这些 token 在中文输入里属于专有名词性质，不算外语成分。
///
/// 规则：
/// - 把输入中匹配 [`ASCII_NEUTRAL_TOKENS`] 的子串先替换为空
/// - 再扫描剩余字符：含 CJK + 含 ASCII 字母 → Mixed；只含 CJK → Zh；只含 ASCII 字母 → En；都没有 → Unknown
#[must_use]
pub fn detect(input: &str) -> Language {
    let scrubbed = scrub_neutral_tokens(input);

    let mut has_cjk = false;
    let mut has_ascii_letter = false;

    for ch in scrubbed.chars() {
        if is_cjk(ch) {
            has_cjk = true;
        } else if ch.is_ascii_alphabetic() {
            has_ascii_letter = true;
        }
        if has_cjk && has_ascii_letter {
            return Language::Mixed;
        }
    }

    match (has_cjk, has_ascii_letter) {
        (true, true) => Language::Mixed,
        (true, false) => Language::Zh,
        (false, true) => Language::En,
        (false, false) => Language::Unknown,
    }
}

/// 在 zh + ASCII 混合判定前，把这些 token 视为"中性词"（不算外语成分）。
/// 范围严格控制：文件扩展名 / 大小单位 / 一组高频技术专名。
/// 添加新词需先在 evals 上量化是否会把真实 mixed query 误判为 zh。
const ASCII_NEUTRAL_TOKENS: &[&str] = &[
    // 文件扩展名 / 文件类型词
    "ppt", "pptx", "doc", "docx", "xls", "xlsx", "pdf", "txt", "md",
    "zip", "rar", "mp4", "mp3", "mov", "avi", "png", "jpg", "jpeg",
    "markdown", "excel", "word", "powerpoint",
    // 大小单位（含数字粘连后的纯字母部分；数字会被独立跳过）
    "kb", "mb", "gb", "tb",
];

fn scrub_neutral_tokens(input: &str) -> String {
    let mut s = input.to_lowercase();
    // 先按长 token 顺序替换避免短词覆盖长词
    let mut tokens: Vec<&&str> = ASCII_NEUTRAL_TOKENS.iter().collect();
    tokens.sort_by_key(|t| std::cmp::Reverse(t.len()));
    for t in tokens {
        s = s.replace(t, " ");
    }
    // 再把"100mb"残留的纯数字 + 残留空白都过掉
    s.chars().filter(|c| !c.is_ascii_digit()).collect()
}
```

- [ ] **Step 3: Implement regex-based target_ref extraction**

把 `packages/intent-parser/src/lib.rs` 的 `extract_target_ref` 函数（line ~315）替换为：

```rust
fn extract_target_ref(input: &str, lower: &str) -> Option<TargetRef> {
    use regex::Regex;
    use std::sync::OnceLock;

    // 1. 数字紧跟"第" — "第1个" / "第 1 个" 都可
    static RE_ZH_DIGIT: OnceLock<Regex> = OnceLock::new();
    let re_zh_digit = RE_ZH_DIGIT.get_or_init(|| Regex::new(r"第\s*(\d+)\s*个").expect("regex"));
    if let Some(cap) = re_zh_digit.captures(input) {
        if let Ok(value) = cap[1].parse::<u32>() {
            return Some(TargetRef::LastResults { selector: TargetSelector::Index { value } });
        }
    }

    // 2. 中文 ordinal word
    let cn_ord = match () {
        () if input.contains("第一个") => Some(1u32),
        () if input.contains("第二个") => Some(2),
        () if input.contains("第三个") => Some(3),
        () if input.contains("第四个") => Some(4),
        () if input.contains("第五个") => Some(5),
        () => None,
    };
    if let Some(value) = cn_ord {
        return Some(TargetRef::LastResults { selector: TargetSelector::Index { value } });
    }

    // 3. 英文 "the N result" / "the N-th"
    static RE_EN_DIGIT: OnceLock<Regex> = OnceLock::new();
    let re_en_digit = RE_EN_DIGIT.get_or_init(|| {
        Regex::new(r"\bthe\s+(\d+)(?:st|nd|rd|th)?\s+(?:result|one|item|file)\b").expect("regex")
    });
    if let Some(cap) = re_en_digit.captures(lower) {
        if let Ok(value) = cap[1].parse::<u32>() {
            return Some(TargetRef::LastResults { selector: TargetSelector::Index { value } });
        }
    }

    // 4. 英文 ordinal word
    let en_ord = match () {
        () if lower.contains("first one") || lower.contains("the first") => Some(1u32),
        () if lower.contains("second one") || lower.contains("the second") => Some(2),
        () if lower.contains("third one") || lower.contains("the third") => Some(3),
        () if lower.contains("fourth one") || lower.contains("the fourth") => Some(4),
        () => None,
    };
    en_ord.map(|value| TargetRef::LastResults { selector: TargetSelector::Index { value } })
}
```

```rust
        .iter()
        .any(|s| lower.contains(s));
    let clear_signal = ["不限制", "lift the limit", "clear the location", "clear the time"]
        .iter()
        .any(|s| lower.contains(s));
    let limit_to_signal = ["limit to", "限定到", "限定为"].iter().any(|s| lower.contains(s));

    if !(only_signal || exclude_signal || sort_signal || clear_signal || limit_to_signal) {
        return None;
    }

    let mut delta = RefineDelta::default();
    let mut clear: Option<Vec<locifind_search_backend::ClearableField>> = None;

    if only_signal {
        if let Some(m) = match_extensions(lower) {
            if !m.extensions.is_empty() {
                delta.extensions = Some(m.extensions.iter().map(|s| (*s).to_string()).collect());
            }
            delta.file_type = Some(m.file_type);
        }
        if let Some(loc) = parse_location_with_language(lower, language) {
            delta.location = Some(loc);
        }
        // 时间："只看上周修改的" / "only in last week"
        let (created, modified, accessed) = parse_time_fields(lower, input);
        if created.is_some() { delta.created_time = created; }
        if modified.is_some() { delta.modified_time = modified; }
        if accessed.is_some() { delta.accessed_time = accessed; }
    }

    if limit_to_signal {
        let (created, modified, accessed) = parse_time_fields(lower, input);
        if created.is_some() { delta.created_time = created; }
        if modified.is_some() { delta.modified_time = modified; }
        if accessed.is_some() { delta.accessed_time = accessed; }
    }

    if exclude_signal {
        if let Some(m) = match_extensions(lower) {
            delta.exclude_file_type = Some(vec![m.file_type]);
        }
    }

    if sort_signal {
        if lower.contains("按大小") || lower.contains("by size") {
            delta.sort = Some(SortOrder::SizeDesc);
        } else if lower.contains("按名称") || lower.contains("by name") {
            delta.sort = Some(SortOrder::NameAsc);
        }
    }

    if clear_signal {
        // "不限制 X" 中默认是 location；"clear the time limit" 走 time
        let field = if lower.contains("time") || lower.contains("时间") {
            locifind_search_backend::ClearableField::ModifiedTime
        } else {
            locifind_search_backend::ClearableField::Location
        };
        clear = Some(vec![field]);
    }

    let _ = input;

    Some(SearchIntent::Refine(Refine {
        schema_version: SchemaVersion::V1,
        language: Some(language),
        base_ref: BaseRef::LastIntent,
        delta,
        clear,
    }))
}
```
