# LociFind 使用手册

> **Local search for humans** —— 用大白话查电脑里的文件，不用记任何搜索语法。
> 本手册随程序一起分发。第一次用，从「3 分钟上手」开始就够了。

---

## 1. 这是什么

LociFind 是一个**本地优先**的电脑文件搜索工具。你用**自然语言**（说人话）描述要找什么，它替你去问系统已有的搜索索引（Windows Search / Everything），把结果列出来。

一句话区别：**Windows 自带搜索和 Everything 都只认关键词，LociFind 让你能直接说「上周改过的 ppt」「下载目录里最大的几个文件」这种带条件的话。**

**重要：一切都在你本机完成。** 不联网、不上传你的文件名、路径、内容或搜索词。

---

## 2. 安装与启动

你拿到的可能是两种形式之一：

### 形式 A：安装包（`LociFind_x.x.x_x64-setup.exe`）
1. 双击运行。
2. **会弹 Windows 蓝色拦截框**（"Windows 已保护你的电脑"）——这是因为程序还没买代码签名证书，**不是病毒**。点 **「更多信息」→「仍要运行」** 即可。
3. 按提示装完，从开始菜单打开 **LociFind**。

### 形式 B：免安装版（解压后的文件夹）
1. **整个文件夹一起解压**，不要只把 `LociFind.exe` 单独拖出来——它旁边的 `synonyms` 等文件夹是同义词词典，缺了同义词功能会降级。
2. 双击文件夹里的 `LociFind.exe`。
3. 同样可能弹蓝色拦截框，处理方式同上。

### 前置条件
- **Windows 10 / 11，64 位。**
- 需要 **WebView2 运行时**（Win11 一般自带；Win10 若提示缺失，安装包会自动装，免安装版需到微软官网搜 "WebView2 Runtime" 装一次）。
- 建议系统的 **Windows Search 索引已开启**（默认就是开的）。
- 可选：装了 **Everything**（voidtools）会让纯文件名搜索更快——没有也能用。

## 3. 3 分钟上手

1. 打开 LociFind，看到一个搜索框，写着「用自然语言描述你要找的文件...」。
2. 输入 `查找 pdf`，按**回车**。
3. 下面会列出结果。每条显示：文件名、完整路径、来自哪个后端（`via search.windows` / `via search.everything` / `via search.local` 本地索引）、修改时间、大小。
4. 顶部有个灰色徽章，显示它**理解成了什么**（intent），比如 `file_search`、命中的信号词、用了哪个后端、耗时多少。

就这么简单：**说一句话，回车，看结果。**

## 4. 你可以这样问（按场景）

直接把下面的句子抄进搜索框即可，中英文都支持。

### 场景一：按类型 / 关键词找文件
| 你可以说 | 它会做 |
|---|---|
| `查找 pdf` / `find pdf` | 列出所有 PDF |
| `找 ppt` / `查找 word 文档` | 按类型找 |
| `查找包含 报告 的文档` | 文件名 / 内容含「报告」的文档 |
| `找叫 预算 的文件` | 名字含「预算」的文件 |
| `find files containing invoice` | 含 invoice 的文件 |

### 场景二：带时间、大小、数量的组合条件（这是 LociFind 最有用的地方）
| 你可以说 | 它会做 |
|---|---|
| `查找昨天编辑过的 ppt` | 限定修改时间 |
| `找最近一周的 pdf` | 近 7 天 |
| `find files modified this month` | 本月修改的 |
| `找最大的 10 个视频` | 按大小排序，取前 10 |
| `查找大于 100MB 的文件` | 按大小过滤 |
| `最新的 5 个文档` | 按时间排序取前 5 |

### 场景三：限定在某个目录
| 你可以说 | 它会做 |
|---|---|
| `找下载目录里的 pdf` | 只在「下载」里找 |
| `查找桌面上的图片` | 只在「桌面」 |
| `find pdf in documents` | 只在「文稿 / 文档」 |

### 场景四：找图片、视频、音乐
| 你可以说 | 它会做 |
|---|---|
| `找截图` / `find screenshots` | 找截图 |
| `查找 jpg 图片` | 按图片格式 |
| `查找时长超过 3 分钟的视频` | 按视频时长 |
| `find mp4 videos` | 找 MP4 |

> 注：音乐的歌手 / 专辑等标签信息，Windows Search 支持有限，能搜文件名但不一定能精确按歌手匹配。

### 场景五：同义词智能扩展（LociFind 的特色）
你不用想起文件的确切叫法。比如你搜 **`工作汇报`**，它会自动把「述职」「工作总结」「年度总结」等近义说法一起搜，所以一个叫 `述职2024.docx` 的文件也能被你用「工作汇报」找到。

| 你可以说 | 可能命中 |
|---|---|
| `工作汇报` | `述职.pptx`、`年度总结.docx` 等 |
| `合同` | `购房协议.pdf` 等（协议≈合同） |
| `找截图` | `屏幕截图_xxx.png`（截图≈截屏≈屏幕截图） |

## 5. 进阶：多轮细化 + 文件操作

LociFind 记得你**上一次的搜索结果**，可以接着操作。

### 5.1 逐步收窄（细化）
先 `查找 pdf`，再输入下面任一条，会**在上一轮基础上继续过滤**：
- `只看下载目录`
- `只要最近三天的`
- `只看大于 1MB 的`

### 5.2 打开 / 定位（立即执行，无需确认）
先搜索拿到列表，再：
- `打开第1个` / `打开第二个` —— 用默认程序打开
- `在资源管理器里显示第3个` —— 跳到文件所在文件夹并选中
- `open the 2nd one` —— 英文也行

### 5.3 复制 / 移动 / 重命名（会真改文件，**必须确认**）
这类操作会**弹确认框，点「确认」才执行，点「取消」文件不动**：
- `把第1个复制到桌面`
- `把第2个移动到下载`（注意写法：`把第N个移动到X`）
- `把第1个重命名为 新名字`

支持的目标目录：**桌面 / 下载 / 文稿 / 图片**。

> ⚠️ **安全提示**：
> - **删除功能被完全禁用**，LociFind 不会、也无法删你的文件。
> - 复制 / 移动 / 重命名前请确认看清确认框内容。测试时建议先拿不重要的文件练手。

## 6. 和其他搜索工具的区别

| 对比项 | Windows 自带搜索 | Everything | **LociFind** |
|---|---|---|---|
| 搜文件名 | 慢、常搜不全 | **极快**，但只搜文件名 | 借用二者，按需路由 |
| 搜文件内容 | 支持（索引内） | 不支持 | 支持（走 Windows Search） |
| 说人话查（带时间/大小/条件）| ❌ 只认关键词 | ❌ 只认关键词 + 符号语法 | ✅ **自然语言** |
| 同义词智能扩展 | ❌ | ❌ | ✅ |
| 多轮细化 + 找到就操作 | ❌ | ❌ | ✅（打开/定位/复制/移动/重命名）|
| 隐私（是否上云）| 本地 | 本地 | **本地，不联网** |
| AI 大模型（如 Copilot）| 需新硬件、部分上云 | — | 不绑硬件、不上云 |

**一句话总结**：Everything 快但只认文件名；Windows 搜索能搜内容但不会说人话；系统 AI 要新硬件还可能上云。LociFind 把「说人话 + 本地 + 已有索引」三者合在一起。

## 7. 常见问题

**Q：打开时被 Windows 蓝框 / 杀软拦了？**
A：因为程序还没买代码签名证书（测试版正常现象）。点「更多信息 → 仍要运行」即可。这不影响功能、不是病毒。

**Q：搜不到明明存在的文件？**
A：多半是 **Windows Search 还没把那个目录索引进去**。检查：设置 → 搜索 → 搜索 Windows → 看该目录在不在索引范围。或装 Everything 走文件名直搜。

**Q：中文文件名显示成乱码？**
A：本测试版已专门修复了中文文件名编码问题。如果仍遇到，请记下你搜的那句话反馈给我。

**Q：第一次搜索有点慢？**
A：首次启动要初始化后端，之后会快。如果某次搜索超时，重试一下（系统索引偶尔会抽风）。

**Q：会偷偷传我的文件 / 搜索记录吗？**
A：不会。LociFind 默认完全本地运行，不联网。

**Q：怎么彻底卸载？会留垃圾吗？**
A：安装版直接走系统「设置 → 应用 → 卸载」即可——卸载器会自动删掉本机索引、已下载的模型、日志、审计记录、搜索历史和你的同义词库，只保留设置文件（重装后配置仍在）；版本升级不会触发清理。免安装版（或想在卸载前手动清）：打开「隐私」页 → 「卸载清理」→ 确认，然后删掉程序文件夹即可。
